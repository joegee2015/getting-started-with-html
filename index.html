Computers are Stupid ?

Programmers have to be careful writing codes exactly the computer can understand, with correct syntax etc.

For example, if we forget to close a tag, the computer won't be able to figure out what we had intended to make italicize. This "stupidity" can be very frustrating, and it's a human error for not putting the closing tag.
